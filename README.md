# Shortest Path Visualizer

A browser-only web app that visualizes shortest-path algorithms (BFS, Dijkstra, A*, Greedy Best-First, Bellman-Ford) on a static world city graph.

## Features
- **5 Algorithms:** Visualize BFS, Dijkstra, A*, Greedy Best-First, and Bellman-Ford.
- **Interactive Map:** Built with Leaflet and CartoDB Dark Matter tiles.
- **Real-world Dataset:** Over 500 cities and connections globally.
- **Live Stats:** View calculation time, nodes visited, distance, and total steps.
- **Adjustable Speed:** Control animation speed or skip instantly to the result.
- **Shareable State:** URL parameters to share specific algorithm runs between cities.

## Tech Stack
- Vanilla JavaScript (ES Modules)
- Vite for building
- Tailwind CSS (via CDN)
- Leaflet.js for mapping
